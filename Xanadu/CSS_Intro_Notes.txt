-- CSS defined --

  Cascading Style Sheets
  A styling language developed alongside HTML
  Important in separating the concerns of content and design
  Enables styling of every viewable element on a page along with animations (complex animation normally done with JS)

-- Basic Syntax --

    (selector here normally meaning element, class, or id)

    selector {
        property: value;
    }

    {this is the declaration block}


-- Inline / Embedded / External style sheet --

    Inline   - CSS style written via style attributes on HTML elements (least scalable)
    Embedded - CSS written in style tags (ok for temporary experimenting)
    External - CSS written in separate style sheet and linked to the HTML via a link element (best practice)

    External styling is overridden by embedded styling which in turn is overridden by inline styling
